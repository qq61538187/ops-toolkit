快速开始

启动服务

# 启动 Docker Compose 服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs nginx

📁 目录结构

  • conf.d/ - 站点配置
  • html/ - 网站文件
  • ssl/ - SSL 证书
  • nginx.conf - 主配置

🌐 访问地址

  • HTTP: localhost:80
  • HTTPS: localhost:443
  • 健康检查: /health
